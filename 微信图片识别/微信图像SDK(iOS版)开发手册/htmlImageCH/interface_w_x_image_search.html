- (void) setAppID: (NSString *)  appid

设置APPID，使用前设置，有可能会进行联网认证

appid微信公众平台的APPID
+ (WXImageSearch *) sharedImageSearch

每次在需要使用图像识别前，可通过此+方法获取单例对象

返回
返回WXImageSearch对象
- (BOOL) startWithImage: (UIImage *)  image

开始识别图像，只有本次识别结束（成功、识别、取消）后，才可以开始下一次识别

参数
image要识别的图片，UIImage对象，大图会被缩放到160000像素
返回
YES：已开始识别，NO：失败原因请参加imageSearchMakeError（如果上次识别还未结束，则不产生错误码）
- (BOOL) startWithJPGImageData: (NSData *)  imageData

开始识别图像，只有本次识别结束（成功、识别、取消）后，才可以开始下一次识别

参数
imageData要识别的图像数据，JPG格式，必须限制在300 000字节以内
返回
YES：已开始识别，NO：失败原因请参加imageSearchMakeError（如果上次识别还未结束，则不产生错误码）
