- (BOOL) startOnce

开始一次语音识别。录音并进行识别，当长时间静默时（默认设置为：开始5秒仍没有说话，或者说话后1.5秒）录音会停止。

返回
YES 开始进行识别；NO 开始失败，失败信息见回调消息voiceInputMakeError，如果上次识别没有结束而试图再次开始，返回值为NO且不会回调错误码
